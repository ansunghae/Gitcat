## 사용방법 - 직접 호스팅하여 사용

1. Node.js 16.15.1 이상을 설치합니다.
2. 이 레포지토리를 내려받습니다.
3. ``npm install``를 터미널(또는 CMD)에 입력해 모듈을 설치합니다.
4. 레포지토리를 설치한 경로로 들어간 후 ``node index.js``를 입력해 봇을 실행합니다.
